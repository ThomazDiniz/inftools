# Título Tamanho Agendado para Status
Como configurar o YouTube Scheduler
1
Crie um projeto no Google Cloud
Acesse o Google Cloud Console, clique em "Selecionar projeto" no topo e depois "Novo projeto". Dê um nome qualquer (ex: inftools).
2
Ative a YouTube Data API v3
No menu lateral, vá em APIs e Serviços → Biblioteca. Busque por "YouTube Data API v3" e clique em Ativar.
3
Crie uma credencial OAuth 2.0
Vá em APIs e Serviços → Credenciais → Criar credencial → ID do cliente OAuth 2.0. Selecione o tipo Aplicativo da Web.
4
Adicione sua origem autorizada
Em Origens JavaScript autorizadas, adicione a URL onde você vai usar a tool. Ex: https://seuuser.github.io. Para testar localmente, adicione também http://localhost.
5
Configure a tela de consentimento (se necessário)
Se solicitado, vá em Tela de consentimento OAuth, escolha tipo Externo e preencha o nome do app. Não precisa publicar — modo de teste é suficiente. Adicione seu e-mail como usuário de teste.
6
Cole o Client ID aqui
Após criar a credencial, copie o Client ID (formato xxxxx.apps.googleusercontent.com) e cole no campo da sidebar. Ele fica salvo no seu navegador. Nenhum client_secret é usado — o login acontece via popup do Google, igual ao fluxo normal de "Entrar com Google".
